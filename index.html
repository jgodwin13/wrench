README

wrench is a command-line interpreter that can run on Windows 8.1, Windows 8, Windows 7, Windows Vista, Windows XP, Windows NT, Windows CE, OS/2 and eComStation operating systems.

What is this repository for?

Quick summary

wrench provides the daily windows command-line(cmd.exe)user with a UNIX command-line feel with its command references similar to those of a typical UNIX terminal.
